# Contexto del Proyecto

## Descripción general
Planchado Express es un sistema automatizado diseñado para optimizar el proceso de planchado de prendas mediante la integración de componentes mecánicos, electrónicos y de control. El proyecto busca reducir la intervención manual del usuario, mejorar la eficiencia del proceso y garantizar resultados consistentes en el acabado de las prendas.

El sistema se concibe como un prototipo dentro del área de los sistemas ciberfísicos, en donde se combinan elementos físicos (estructura, actuadores, transferencia de calor) con una capa de control y toma de decisiones basada en programación.

## Funcionamiento general
El sistema realiza un proceso de planchado asistido mediante una secuencia automatizada que incluye:

1. Recepción o colocación de la prenda.
2. Aplicación de calor controlado.
3. Ejecución de presión o desplazamiento sobre la tela.
4. Finalización del proceso con mínima intervención del usuario.

Estas acciones permiten estandarizar el proceso de planchado y disminuir la variabilidad asociada al uso manual.

## Tipo de prendas
El prototipo está enfocado principalmente en el tratamiento de prendas de tela plana, tales como:

* Camisas
* Playeras
* Prendas ligeras de uso cotidiano

Esto permite validar el comportamiento del sistema en condiciones controladas y con materiales comunes.

## Alcance del proyecto
El desarrollo corresponde a un prototipo funcional, orientado a demostrar la viabilidad técnica del sistema y la integración de sus componentes principales. No se busca aún un producto comercial terminado, sino una solución experimental capaz de validar el concepto de automatización del planchado.

## Nivel de automatización
El sistema presenta un nivel de automatización semiautomático a automático, ya que ejecuta tareas clave del proceso de planchado de forma autónoma, aunque puede requerir intervención del usuario en etapas iniciales o de supervisión.

## Idea principal
Desarrollar un sistema ciberfísico capaz de automatizar el proceso de planchado, reduciendo el esfuerzo humano y mejorando la eficiencia y consistencia del resultado.
